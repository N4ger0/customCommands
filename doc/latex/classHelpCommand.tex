\doxysubsection{Detailed Description}
A concrete implementation of the {\ttfamily \doxylink{classCommand}{Command}} class for displaying help information. 

The {\ttfamily \doxylink{classHelpCommand}{Help\+Command}} displays usage instructions and a list of available commands within the parser.
